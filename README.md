# Run in Command

> **1. Install  Dependencies**
>``` bash
>npm install
>```

> **2. Run app**
>``` bash
>npm start
>```

> **API Endpoint**
>``` bash
> http://localhost:8080/tax?monthlySalary={monthlySalary}&year={year}
>```

> **Query params**
>``` bash
> monthlySalary = integer
> year = integer
>```
